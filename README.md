## 🚀 Overview

This repository contains my personal implementation of Linked Lists from scratch. Moving away from standard built-in libraries like `java.util.LinkedList`, this project focuses on the core mechanics of node manipulation, memory management, and pointer traversal.

### ✨ Key Operations Implemented:
* **Insertion:** Add nodes at the Beginning, End, or a Specific Index.
* **Deletion:** Remove nodes from the Beginning, End, or by Value/Index.
* **Traversal:** Print and iterate through the list efficiently.
* **Search:** Find specific elements and return their positions.
* **Advanced (Work in Progress):** Reversing the list, detecting cycles, and finding the middle node.

---

## 🧠 Time & Space Complexity

Understanding the efficiency of these operations is crucial for competitive programming:

| Operation | Time Complexity | Space Complexity |
| :--- | :---: | :---: |
| **Access (Index)** | $O(n)$ | $O(1)$ |
| **Search (Value)** | $O(n)$ | $O(1)$ |
| **Insert (Head)** | $O(1)$ | $O(1)$ |
| **Insert (Tail)** | $O(n)$ * | $O(1)$ |
| **Delete (Head)** | $O(1)$ | $O(1)$ |
| **Delete (Node)** | $O(n)$ | $O(1)$ |

*\* Note: Insertion at the tail is $O(1)$ if a `tail` pointer is maintained, otherwise $O(n)$.*
